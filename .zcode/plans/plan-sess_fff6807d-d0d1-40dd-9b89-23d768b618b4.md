# 🔄 诗词名言模块 — 改为关键词搜索模式

## 核心改动

### 用户流程（改后）
```
用户输入 "思念" → 系统同时搜索诗词库 + 歌词库
                  → 输出：📜 匹配诗词 (3首) + 🎵 匹配歌词 (4首)
                  → 每条可复制 / 仿写
```

---

## 🛠️ 实施

### 1. 数据层 — 给每条加 keywords 字段
**POETRY_DB 每条的 keywords：**
```js
{ text:'思君如满月，夜夜减清辉。', ..., keywords:['思念','相思','深情','离别','等待'] }
{ text:'长风破浪会有时，直挂云帆济沧海。', ..., keywords:['励志','信念','奋斗','坚持'] }
```

**LYRICS_DB 每条的 keywords：**
```js
{ lyric:'思念是一种很玄的东西，如影随形。', ..., keywords:['思念','爱情','深情','执着'] }
```

### 2. 搜索算法
- 匹配 `text` 包含关键词 OR `author` 包含 OR `keywords` 包含
- 诗词和歌词分开搜索，分别展示

### 3. UI 改动
```
📖 诗词名言
[🔍 输入主题/关键词...]          ← 搜索框放在最上面

无搜索时 → 显示浏览模式（保持现有三个 tab）
有搜索时 → 显示 "📜 匹配诗词 (X首)" + "🎵 匹配歌词 (X首)"
           → 每组显示 2-3 条，可展开看更多
```

### 4. 保留功能
- ✨ 仿写原创（诗词条目）
- 📋 复制
- ❤️ 收藏
- 🏮/✒️/💡 分类浏览（无搜索时）
